# Web

React frontend for the portfolio site.

## Local development

Install dependencies from the repo root (only needed once):

```bash
cd ~/code/personal/portfolio-website
npm install
```

Start the dev server:

```bash
cd ~/code/personal/portfolio-website/src/web
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

The dev server hot-reloads on save — no manual refresh needed.
